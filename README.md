# Basic JavaScript Projects

A collection of interactive JavaScript projects demonstrating fundamental web development concepts including DOM manipulation, event handling, state management, and game logic. Perfect for learning and practicing JavaScript, HTML, and CSS.

## Projects Overview

This repository contains three beginner to intermediate-level JavaScript projects that showcase different aspects of web development:

1. **Currency Converter** - Real-time currency conversion with live exchange rates
2. **Rock Paper Scissors Game** - Interactive game with visual assets and score tracking
3. **Tic Tac Toe** - Two-player game with intelligent game logic and win detection

### Project-Specific Features

#### Currency Converter
- Real-time exchange rate conversion
- Support for multiple currencies
- Interactive input fields
- Reverse currency conversion functionality
- Clean and intuitive interface

#### Rock Paper Scissors Game
- Visual game assets for better user experience
- Username personalization
- Score tracking and statistics
- Instant game result feedback
- Play multiple rounds without page refresh

#### Tic Tac Toe
- Two-player gameplay
- Win/Draw detection
- Game reset functionality
- Visual game board with clear state
- Move validation
